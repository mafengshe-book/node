# 码蜂社 Node.js 讲义

> 基于 gitbook 构建的前端讲义

## 本地部署方法

### 克隆此仓库
```
git clone https://github.com/mafengshe-book/node.git
```
### 安装依赖

```
npm install
```

### 运行

```
npm start
```

### 构建

```
npm run build
```

### 部署

```
npm run deploy
```
部署前请先**构建**
